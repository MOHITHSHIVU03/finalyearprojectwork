PROJECT PHASES

CLASSFICATION OF A DOCUMENT

The input text document is taken from the user and in this phase we classify whether the document is Sensitive or Non-Sensitive. We use the sensitive words in the document like diseases or drugs etc to classify the document. If the document is sensitive it is further moved to the next phase i.e either the Generalization or Sanitization phase. If the document is Non-Sensitive it is sent to the Suppression phase.


GENERALIZATION

Generalization is a privacy preserving approach in which it deliberately removes some of the data to make it less identifiable. Data can be modified into a set of ranges or a broad area with appropriate boundaries. You can remove the house number in an address, but make sure you don’t remove the road name. The purpose is to eliminate some of the identifiers while retaining a measure of data accuracy.


SANITIZATION

The PIDs such as Explicit and Quazi Identifiers are extracted and then removed and replaced by Synthetic data respectively and thus privacy preservation of patient's information is achieved partially.


SUMMARIZATION

In this step, we summarize the document. Summarization is the process of highlighting medical information which helps medical experts in efficiently identifying the records. Extractive summaries are created by borrowing phrases or sentences from the original input text. The user is given an option to download the summarized document in which clinical terms such as dosage, drug, duration, form, frequency, route, and strength are highlighted for better visual appearance.
